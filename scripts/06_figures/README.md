### Figures and tables

This directory contains the R scripts used to generate the main figures, supplementary figures, and manuscript tables.

#### Structure

- `Main_figures/`  
  R scripts for main manuscript figures

- `Supplementary_figures/`  
  R scripts for supplementary figures

- `Tables/`  
  R scripts used to generate manuscript tables

#### Naming convention

Scripts are named according to the figure or table they generate. For example:

- `Figure_2_UMAP.R`
- `Figure_S3_heatmap_key_markers.R`
- `Table_S1.R`

This naming scheme is intended to make it easy to match each script to the corresponding figure or table in the manuscript.

#### Notes

- Input objects and intermediate files are expected to be generated in earlier steps of the pipeline.
